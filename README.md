# AlgeoMath 클론코딩

알지오메스(AlgeoMath)의 핵심 기능 — 기하 작도, 대수창 연동, 종속성 엔진 — 을 단계별로 재현하는 프로젝트입니다.

**배포 주소**: [https://kki147.github.io/algeomath/](https://kki147.github.io/algeomath/)

> **진행 방식**: 이 README의 로드맵 순서대로 구현합니다.  
> 완료 여부는 [`task.md`](task.md) 에서 체크하며, 세부 설계는 [`implementation_plan.md`](implementation_plan.md) 를 참고합니다.

---

## 기술 스택

| 항목 | 선택 |
|------|------|
| 렌더링 | HTML5 Canvas |
| UI / 이벤트 | jQuery |
| 스타일 | Vanilla CSS (Flexbox) |
| 언어 | ES5 호환 JavaScript (화살표 함수 미사용) |
| 호환성 | IE 포함 레거시 브라우저 |

`common/` 폴더는 공용 라이브러리이므로 **수정하지 않습니다.**

### 아키텍처 (3계층)

```text
┌─────────────────────────────────────────┐
│  AlgeoApp        — 도구·마우스·대수창 UI  │
├─────────────────────────────────────────┤
│  AlgeoEngine     — 객체 모델 + DAG 종속성 │
├─────────────────────────────────────────┤
│  AlgeoRenderer   — 좌표변환·격자·객체 렌더 │
└─────────────────────────────────────────┘
```

| 모듈 | 역할 |
|------|------|
| `AlgeoEngine` | Point, Segment, Circle 객체 관리. 부모-자식 DAG로 종속성 전파·연쇄 삭제 |
| `AlgeoRenderer` | 수학 좌표 ↔ 화면 좌표 변환, Pan/Zoom, 격자·축·객체 그리기 |
| `AlgeoApp` | 도구바, 캔버스 이벤트, 대수창 갱신, 수식 파서 |

## 실행 방법

1. `index.html` 을 브라우저에서 엽니다. (로컬 서버 불필요)
2. `common/js/popscale.js` 가 `#wrap` 스케일을 적용한 뒤 캔버스가 초기화됩니다.
3. 상단 도구바와 좌측 대수창, 우측 작도 영역이 표시되면 정상입니다.

## 로드맵 (순차 진행)

각 단계는 **이전 단계 완료 후** 다음 단계로 넘어갑니다.

---

### 1단계 — 작도 도구 및 엔진 ✅ 완료

기하 객체를 마우스로 작도하고, 종속 관계에 따라 함께 움직이는 핵심 엔진을 구축합니다.

| # | 항목 | 상태 |
|---|------|------|
| 1-1 | UI 레이아웃 (도구바 · 대수창 · 캔버스) | ✅ |
| 1-2 | 좌표 변환, Pan/Zoom, 격자·축 렌더링 | ✅ |
| 1-3 | AlgeoEngine — Point / Segment / Circle + DAG | ✅ |
| 1-4 | 도구 연동 — 이동, 점, 선분, 원, 삭제 | ✅ |
| 1-5 | 점 드래그 시 종속 선분·원 실시간 갱신 | ✅ |
| 1-6 | 작도 중 선택 점 강조, 객체 이름 규칙 (AB, ⊙A) | ✅ |
| 1-7 | 대수창 객체 리스트 동적 갱신 | ✅ |

**도구 사용법**

| 도구 | 조작 |
|------|------|
| 이동 | 빈 곳 드래그 → Pan / 점 드래그 → 이동 |
| 점 | 캔버스 클릭 → 점 생성 (A, B, C …) |
| 선분 | 점 2개 순서대로 클릭 → 선분 AB |
| 직선 | 점 2개 순서대로 클릭 → 무한 직선 dAB (남색 점선) |
| 중점 | 점 2개 클릭 → 중점 MAB (보라색, 드래그 불가) |
| 수직이등분선 | 점 2개 클릭 → 수직이등분선 pbAB (청록 점선) |
| 원 | 중심점 → 둘레 점 클릭 → 원 ⊙A |
| 삭제 | 점·선분·직선·원 클릭 → 연쇄 삭제 |

---

### 2단계 — 대수창 수식 입력 및 파서 🔄 진행 중

대수창에 텍스트를 입력해 객체를 생성·수정합니다.

| # | 항목 | 상태 |
|---|------|------|
| 2-1 | 입력 UI (입력창, 버튼, Enter, 오류 메시지) | ✅ |
| 2-2 | `A = (1, 2)` 점 좌표 파싱 (생성 / 이동) | ✅ |
| 2-3 | `y = 2x + 1` 일차함수 그래프 | ✅ |
| 2-4 | `AB` 선분, `⊙(A,B)` 원 등 기하 객체 직접 정의 | ✅ |
| 2-5 | 대수창 항목 클릭 → 캔버스 객체 하이라이트 | ✅ |

**2-4 구현 완료**

- `AB` — 기존 점 A, B로 선분 생성 (이미 있으면 중복 생략)
- `⊙(A, B)` — 중심 A, 둘레 점 B로 원 생성 (이름: `⊙A`)
- `⊙AB` — `⊙(A, B)` 축약형

**2-3 구현 완료**

- `FUNCTION` 객체 타입 (slope, intercept, expression)
- `y = ax + b` 파싱 (`2x+1`, `-x`, `5` 등 지원)
- 뷰포트 x범위에 맞춘 보라색 곡선 렌더링
- 대수창에 `f: y = 2x + 1` 형태 표시, 삭제 도구로 곡선 제거 가능

### 3단계 — 고급 작도 도구 🔄 진행 중

| # | 항목 | 상태 |
|---|------|------|
| 3-1 | 직선 (두 점 지나는 무한 직선) | ✅ |
| 3-2 | 중점, 수직이등분선 | ✅ |
| 3-3 | 평행선, 수직선 | ⬜ **다음** |
| 3-4 | 각도, 호(Arc) | ⬜ |
| 3-5 | 다각형 | ⬜ |

**3-1 구현 완료**

- 도구바 `↔` 직선 도구 — 점 2개 클릭으로 무한 직선 작도
- 남색 점선으로 표시 (선분과 구분)
- 점 이동 시 직선 실시간 갱신 (DAG 종속성)
- 대수창 입력: `Line(A, B)` / 이름 `dAB`

**3-2 구현 완료**

- `M` 중점 도구 — 두 점의 중점 (보라색, 종속 점)
- `⊥` 수직이등분선 도구 — 청록 점선, 부모 점 이동 시 갱신
- 대수창: `Midpoint(A,B)`, `PerpBisector(A,B)`
